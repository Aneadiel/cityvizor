Nastavení profilu

Tento název se zobrazuje v záhlaví stránky s profilem obce. Může se lišit od oficiálního názvu obce.
https://cityvizor.cz/profil/
Na této adrese bude dostupný profil vaší obce.
Toto je identifikace vaší obce z databáze RISY, úřední desky a registru smluv. Pokud údaj nesedí, prosíme kontaktujte správce.
Na tomto e-mailu vás budeme kontaktovat, bude-li nějaký problém s vaším profilem.
Pokud zvolíte Veřejný, budou moci návštěvníci portálu CityVizor vidět váš profil, pokud zvolíte Skrytý, profil vaší obce uvidí pouze administrátoři vašeho profilu a administrátoři portálu.
